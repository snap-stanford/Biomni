# Task: Create Boxplot for Group Comparison

Create a boxplot to compare distributions across different groups.

## Input

Data organized by groups:
- Group A: [10.5, 11.2, 10.8, 11.5, 10.9]
- Group B: [9.2, 9.5, 9.0, 9.8, 9.3]
- Group C: [12.1, 11.8, 12.3, 11.5, 12.0]

## Requirements

1. Create a boxplot showing:
   - Box: Interquartile range (IQR)
   - Median line inside box
   - Whiskers: Extend to 1.5Ã—IQR or min/max
   - Outliers: Points beyond whiskers
2. Include:
   - Group labels on x-axis
   - Value labels on y-axis
   - Title
   - Optional: Individual data points (strip plot overlay)
3. Save as PNG file

## Expected Output Format

Provide:
- Boxplot image file (PNG format)
- Brief description of group differences
- Summary statistics for each group
