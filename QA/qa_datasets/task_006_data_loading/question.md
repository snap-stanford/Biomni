# Task: Load Data and Calculate Basic Statistics

Load a TSV file and calculate basic summary statistics.

## Input Data

**File**: `input_data/sample_data.tsv`

A gene expression data file in TSV format.

## Requirements

1. Load the data file: `input_data/sample_data.tsv`
2. Display basic information:
   - Dimensions (number of rows and columns)
   - Column names
   - Data types
3. Calculate and display summary statistics:
   - For numerical columns: mean, standard deviation, min, max
   - For the entire dataset: shape
4. Show the first few rows of the data

## Expected Output Format

Provide a clear summary showing:
- Data dimensions
- Column information
- Summary statistics
- Sample of the data (first few rows)
