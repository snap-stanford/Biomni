# Task: Create Histogram

Create a histogram to visualize the distribution of numerical data.

## Input

A numerical vector or column of data:
```
[10.5, 11.2, 10.8, 11.5, 10.9, 9.2, 9.5, 9.0, 9.8, 9.3, 12.1, 11.8, 10.2, 11.0, 10.6]
```

## Requirements

1. Create a histogram with appropriate binning
2. Include:
   - Clear axis labels
   - Title
   - Appropriate number of bins (or automatic binning)
3. Optionally add:
   - Density curve overlay
   - Mean/median lines
   - Statistics summary
4. Save as PNG file

## Expected Output Format

Provide:
- Histogram image file (PNG format)
- Brief description of the distribution
- Summary statistics (mean, median, std)
