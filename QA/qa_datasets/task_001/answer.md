# BRCA1 유전자 정보

## 1. 전체 이름
**BRCA1 (Breast Cancer 1)** - Breast Cancer Type 1 Susceptibility Protein

## 2. 주요 기능
BRCA1은 종양 억제 유전자로서 다음과 같은 중요한 역할을 합니다:
- DNA 손상 복구 (Homologous recombination repair)
- 세포 주기 조절 (Cell cycle checkpoint control)
- 유비퀴틴화 과정 참여
- 전사 조절 (Transcriptional regulation)

## 3. 관련된 질병
BRCA1 유전자 변이는 다음 질병과 연관됩니다:
- **유방암 (Breast Cancer)**: 유전성 유방암의 약 50%
- **난소암 (Ovarian Cancer)**: 유전성 난소암의 주요 원인
- **전립선암 (Prostate Cancer)**: 남성에서 위험도 증가
- **췌장암 (Pancreatic Cancer)**: 일부 경우 연관

### 위험도
BRCA1 변이 보유자의 경우:
- 유방암 발병 위험: ~70% (70세까지)
- 난소암 발병 위험: ~40% (70세까지)
