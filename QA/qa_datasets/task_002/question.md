# 간단한 통계 분석

다음 데이터에 대한 기본 통계 분석을 수행해주세요:

**데이터**: [10, 20, 15, 25, 30, 18, 22, 28, 16, 24]

다음을 계산하고 보고해주세요:

1. **기본 통계량**
   - 평균 (Mean)
   - 중앙값 (Median)
   - 표준편차 (Standard Deviation)
   - 최솟값, 최댓값

2. **시각화**
   - 히스토그램 생성
   - 박스플롯 생성

3. **결과 해석**
   - 데이터의 분포 특성 설명
   - 이상치(outlier) 여부 확인
